    • Our Products

      Core

      Siminov Core is a open source Object/Relational mapping solution which targets different platforms. It maps an platform specific object mdoel representation to a relational data model (and visa versa).


      It not only take care of the mapping, but also provides data query and retrieval facilities.
      Connect

      Siminov Connect is a lightweight, comprehensive, open source REST framework which is supported on different platforms. It reduces the work load involved in building abd consuming RESTful API's.


      It supports major Internet transport, data format, and service descriptions standards like HTTP and HTTPS, SMTP, XML and JSON.
      Hybrid

      Siminov Hybrid is derived from a combination of technologies, approaches or elements of different kinds. It is an open source framework which leverages best of both native and mobile hybrid technologies.


      It is very difficult to handle database and web service calls. It makes your life easy by mapping JavaScript|Platform specific object model to relational database and also handles all your web services calls using platform container capabilities.
      Studio

      Siminov Studio allows you to speed up your development by targeting different IDE's like (Eclipse, XCode, Visual Studio).


      Using this you can quickly create your app configurations without manually creating it.

An Open Source Framework


All Siminov Framework products are completely Open Source under Apache 2.0 license. Their is no payment required to obtain the build releases - even if you use them in commercial projects, without any usage or feature restrictions. That means that you can use the releases of framework in any app you build without having to pay. We are committed to keeping, maintaining & extending releases of Siminov Framework products.
